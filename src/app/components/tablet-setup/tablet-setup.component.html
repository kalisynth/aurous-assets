Sign in and Preperation

  1. Before beginning this process make sure the device is at least 50% charged and has been added to the Tablet List
  2. Sign in to the device using the device's assigned email example@nacweb.org and the password if you do not know the password ask a Technology Support Officer in Teams or send an email to helpdesk@nacweb.org
  3. Name/Location/Values of settings can be different from device to device, and some devices might not have all the settings listed in this setup guide
  4. Go to device settings and check for updates manually and if there are updates download and install those before continuing, keep repeating this step until the device says that it has the most recent update

Games

  1. if you do not have a copy of TabletSetup.zip please ask a Technology Support Officer in Teams or email helpdesk@nacweb.org
  2. Copy the zip file to the Device Downloads
  3. Extract the zip file to the Downloads folder
  4. Copy the Pictures folder to Internal Storage(where the other 'Pictures' folder is) and overwrite/replace the folder if you are asked to.
  5. Open the Games folder
  6. Install all the apps in the games folder
    1. If it asks if you want to allow from unknown sources make sure to enable that

Data

  1. Open the device Settings
  2. Go to the Connection\Data settings tab
  3. Change the billing cycle range to 26th
  4. Enable and Set the Data limit to 3gb
  5. Set Warning to 2gb

Display

  1. Open the Display Settings Tab
  2. Change Brightness setting to full
  3. Set Screen Timeout to 5 minutes
  4. Change Font Size to Large
  5. Make sure Device is rotated in Landscape and then disable Rotation
  6. Change the Lock Screen to Swipe
  7. Open the Wallpaper app
    1. if there is no Wallpaper app, please open the playstore and download it from there
  8. Select Solid Colors
  9. Tap on the Gray Color
  10. Tap on Set Wallpaper
  11. Tap on Both

Skype

  1. Download/Update/Install Skype
  2. Sign in using the device email address example@nacweb.org
  3. If the account doesnt exist then tap the link that says "or get a new one"
  4. Then create a skype account using the password found in Pins and Passwords and the first name as the device name and the last name as DCVS
  5. Add volunteer1dcvs, volunteer2dcvs, dcvs_coordinator@nacweb.org to the contacts
    1. if setting up a device for a program other then DCVS then just add officedcvs, and refer to that program to find out what other contact you need to add
  6. Open Skype settings and enable auto answer, auto turn on video and accept calls only from contacts
    1. Skype settings can be found in the Chat tab in skype and then tap on the circle with 2 letters, then tap on settings

Aurous

Aurous Plus

  1. Check that you have the latest version of the Aurous App from the playstore
  2. Open the App once it has been installed and click on Google Sign in
  3. pick the example@nacweb.org account
  4. Tap on the 3 dots in the top right corner and go to options
    1. click on unlock aurous services with code
    2. use the pin from pins and passwords
  5. Tap on the Aurous Services button
  6. Tap on add DCVS Contacts

Aurous Exercises

  1. Make sure the exercises have been copied over
  2. Check that Aurous Exercise is installed, if not then install from apk in Tablet Setup Folder
  3. Open Aurous Execises make sure to allow on the permission it asks for

Aurous Services

  1. Check that Aurous Services is installed on the tablet
  2. Open the Settings -> Apps -> Aurous Services
  3. Tap on Apps that can appear on top
  4. Tap on Allow Permission
  5. Tap on Permissons and enable storage

Group

  1. Open Aurous Service App
  2. Tap on Group
  3. Tap on Refresh Groups
  4. Tap on General Group Chat
  5. if it asks to open in Meet or Chrome select Always then select Meet
  6. Tap on Install if it asks you to install
  7. Tap on Continue
  8. When asked to give permission for Audio and Video make sure to allow
  9. if you get an error trying to join a meeting, find GeneralGroup[2].json in the tablet setup folder and move that to the aurousGroups folder and try joining again, if that works delete the other GeneralGroup.json file

NovaLauncher

  1. Open Nova Launcher Settings
  2. Tap on Backup and Import Settings
  3. Tap on Backup
  4. Tap on ok and accept the permissions it asks for
  5. Go to the Tablet Setup folder and copy the AurousNova.novabackup file to 'home folder'/data/com.telsacoilsw.launcher/backup
  6. Go back to Nova Settings -> Backup and Import Settings
  7. Tap on Restore or manage backups
  8. Tap on AurousService -> ok

Find my Device

  1. Open Biometrics and Security settings
  2. Add account
  3. Sign in with google
  4. Tap allow when asked
    5. -- or --
  5. If sign in with google is not available, create an account using the table email and password, first name will be the tablet name and last name will be dcvs, DOB - 01/01/1981
  6. Agree to allow
  7. Wait for the email from samsung then open it and click on the 'verify' button

Accessibility

  1. Download/Update Android Accessibility Suite
  2. Go to Tablet Settings -> Accessibility
  3. Enable the NovaLauncher setting
  4. Enable Select to Speak
  5. Dexterity -> Touch and hold delay -> Custom 5secs

Finish

  1. Go to the tablet list and make sure to update any relevant info
  2. Hand to another person for checking
  3. Finished
